# อิ่มพอดี

แอปคำนวณอาหารตามค่า TDEE พร้อมตัวละครที่โต้ตอบกับพฤติกรรมการกินของคุณ

## ฟีเจอร์ในอนาคต

- คำนวณ TDEE และเป้าหมายแคลอรี่/โปรตีน/ไขมัน/คาร์บ/โซเดียม จากข้อมูลส่วนตัว (เพศ อายุ น้ำหนัก ส่วนสูง ระดับกิจกรรม เป้าหมาย)
- สร้างตัวละครของตัวเอง ปรับทรงผม สีผม สีผิว เสื้อ กางเกง รองเท้าได้
- ตัวละครแสดงอารมณ์ตามการกินในแต่ละวัน: ยิ้มเมื่อทานพอดี พุงป่องเมื่อทานเกิน หิวโซเมื่อทานไม่ครบ
- บันทึกมื้ออาหารด้วยตัวเอง หรือถ่ายรูปอาหาร/บาร์โค้ดให้ AI ช่วยประเมินค่าโภชนาการให้อัตโนมัติ
- นับสตรีควันที่ทานได้พอดีต่อเนื่อง
- ข้อมูลบันทึกอยู่ในเบราว์เซอร์ของตัวเอง (localStorage) ไม่ต้องมีระบบหลังบ้าน

## การใช้งาน

เปิดไฟล์ `impodee.html` ผ่านเบราว์เซอร์ได้เลย หรือจะ deploy ผ่าน GitHub Pages ก็ได้
